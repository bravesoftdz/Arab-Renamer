Arab Renamer is a small tool to convert your Arabic letters to the corresponding
English letters without any translation, but using well known method of writing Arabic.

Main Features:
-Renames Arabic files to the same names but using English characters like the word
'ÞÑÂä'->'Quran'so that files could be used on non-Arabic systems like Linux or on mp3
players.
-limit the files names during renaming.
-using Tashkeel to enhance text convertion( Tested with Quran:download from"
http://www.hdeya.com/~kurtubba/download/?id=Quran.zip")
-The program uses characters like 3, ' and 7 which are popular characters used to 
replace specific Arabic letters:
3=Ú
'=Á
7=Í

Requirements:
- Windows 95, 98, ME, 2000 and XP

Installation:
No installation required.
